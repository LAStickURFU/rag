Правительство расширит функционал портала госуслуг. Будут созданы:
• государственная единая облачная платформа для госорганов;
• национальный государственный центр по выпуску сертификатов безопасности (TSL-сертификатов) для сайтов;
• разделы для купли-продажи автомобилей на портале, при этом договоры будут заверять электронной подписью.
