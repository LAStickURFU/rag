Вводятся новые номера для заполнения кодов налоговых периодов и авансов по НДФЛ ИП на ОСНО.
Что случилось: ФНС России опубликовала законопроект, который устанавливает новые правила заполнения уведомлений для единого налогового платежа. Подробный разбор, как заполнить и подать уведомление об исчисленных суммах на ЕНП, читайте в статье.
Что меняется: теперь при заполнении кодов налоговых периодов «21», «31», «33», «34» для НДФЛ указывается номер:
• 01, 02, 03 — с 1-го по 22-е число первого, второго, третьего месяца квартала соответственно.
• 11, 12, 13 — с 23-го числа по последнее число первого, второго, третьего месяца квартала соответственно.
Например, в январе 2024 года нужно будет указать такие коды:
• 21/01 — с 01.01.2024 по 22.01.2024;
• 21/11 — с 23.01.2024 по 31.01.2024.
Для авансов по НДФЛ для ИП на общей системе налогообложения также предусмотрены новые коды:
• аванс за I квартал — 21/04 (вместо 34/01);
• за полугодие — 31/04 (вместо 34/02);
• за 9 месяцев — 33/04 (вместо 34/03).
