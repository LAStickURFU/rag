По новым правилах правительство будет проверять по плану организации с чрезвычайно высокой и высокой категориями риска и компании-нарушители.
Для какого бизнеса сохранят плановые проверки в 2023:
• юрлица и ИП с категорией чрезвычайно высокого и высокого риска;
• предприятия, оштрафованные за нарушение лицензий;
• предприниматели, которые не устранили нарушения по предписанию.
Как узнать, есть ли организация в графике проверок. Проверить компанию можно по ИНН в сервисе «Поиск проверок» на сайте Генпрокуратуры.
Что с внеплановыми проверками. Проверять внепланово будут, но только по жалобам и с предупреждением о визите инспекторов за сутки. Для налоговой сохранятся и выездные, и камеральные проверки. Аккредитованные IT-компании будут проверять только по поручению руководства ФНС.
