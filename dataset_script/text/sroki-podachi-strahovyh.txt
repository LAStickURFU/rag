Налоговая рассказала, когда предпринимателям подавать заявления на рассрочку по страховым взносам и как формируются суммы по этой рассрочке в 2023 году.
В чём суть рассрочки. В 2022 году ИП и компаниям дали годовую отсрочку по уплате страховых взносов. В 2023 году взносы по этой отсрочке можно оплатить по частям.
Когда подавать заявление о рассрочке:
• с 1 марта по 1 июня — для страховых взносов ИП за 2021 год с дохода свыше 300 тысяч рублей;
• с 1 марта по 28 апреля — для всех остальных отсроченных страховых взносов.
Предприниматель может подать заявление в ИФНС онлайн в электронном виде.
Когда рассчитываются суммы рассрочки. Налоговая автоматически будет определять их ежемесячно 27 числа месяца, на который перенесён срок уплаты. Для организаций и ИП сроки такие:
• 27 мая — по сроку уплаты 29 мая;
• 27 июня — по сроку уплаты 28 июня;
• 27 июля — по сроку уплаты 28 июля;
• 27 августа — по сроку уплаты 28 августа;
• 27 сентября — по сроку уплаты 28 сентября;
• 27 октября — по сроку уплаты 30 октября.
Для ИП с доходом свыше 300 тысяч рублей сумму рассрочки определят 30 июня.
Рассрочку дадут только на перенесённую сумму страховых взносов, суммы взносов для будущих платежей или в составе единого налогового платежа туда не войдут
