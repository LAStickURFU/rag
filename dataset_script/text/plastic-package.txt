Минпромторг представил список упаковки, которая будет под запретом в России. Часть нельзя будет использовать с 2024 года.
Что случилось: Минпромторг направил на согласование перечень из 23 видов упаковки, которые нельзя будет использовать. С 2024 года предлагается запретить шесть видов, остальные 17 будут под запретом с 2030 года.
Что попадёт под запрет:
С 2024 года запретят использовать для упаковки:
• ПЭТ-бутылки всех цветов, кроме голубого, зеленого, коричневого и чёрного;
• непрозрачные ПЭТ-бутылки, за исключением тары белого цвета для молочной продукции;
• упаковку из ПЭТ, произведённую с использованием этикетки из ПВХ;
• многослойные ПЭТ-бутылки;
• капсулы для кофе из многокомпонентных материалов.;
• пластиковые контейнеры для консервов с привальцованной металлической крышкой.
С 2030 года запретят многослойные тубы от зубной пасты, гибкую вакуумную упаковку дойпак, многокомпонентную упаковку Bag-in-Box, сетки для овощей и фруктов и другие виды упаковок.
Что делать бизнесу: использовать для упаковки полимерную продукцию, подходящую для переработки. Например, перейти с непрозрачной ПЭТ-тары на прозрачную, вместо многослойных материалов использовать однослойные.
