Маркетплейс Вайлдберриз запретил продавцам использовать «серые» API для автоматизации рекламных инструментов и аналитики.
Что произошло. 19 и 24 мая маркетплейс дополнил оферту для селлеров (есть в распоряжении Справочной) пунктом 9.10, который регулирует условия использования сайта. В нём запрещается пользоваться сторонними API — инструментом, с помощью которых программы продавца и площадки взаимодействуют друг с другом. За использование «серого», то есть не официального API Вайлдберриз, площадка теперь может заблокировать продавцу аккаунт.
Как это работало раньше. Так как официальное API Вайлдберриз не позволяет создавать рекламные кампании и получать статистику по ней, продавцы использовали сторонние сервисы аналитики и  биддеры.Робот, который умеет управлять контекстной рекламой: принимать решения по настройке и автоматизировать рутинные задачи  Если сервис селлера способен создавать рекламные кампании, пополнять бюджет, смотреть статистику и действия с ключевыми фразами, теперь он будет считаться «серым», и личный кабинет продавца может попасть под блокировку.
Где найти официальный сервис. Доступ предоставляется в личном кабинете, а все условия интеграции опубликованы на официальном сайте с информацией о настройке API на Вайлдберриз.
Точка Маркетплейсы использует для интеграции только официальный API. Узнать подробности и подключить сервис, который поможет управлять продажами, можно по ссылке.
