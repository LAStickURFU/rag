Сервис будет входить в тариф «Премиум», а отдельная подписка будет стоить 9990 рублей в месяц.
Что случилось: Wildberries запустил сервис «Джем» — это подписка на аналитику для продавцов. Сейчас ей может воспользоваться небольшая группа селлеров, но постепенно доступ расширят.
Стоить подписка будет 9990 рублей в месяц, перед покупкой её можно протестировать бесплатно в течение суток. Для владельцев тарифа «Премиум» подписка будет бесплатной.
Что входит в подписку:
• Портрет покупателя. Здесь можно увидеть подробное описание целевой аудитории: пол, возраст, регион, время активности и предпочтения.
• Поисковый отчёт. Селлеру покажут 20 популярных поисковых запросов по каждому товару, их динамику, воронку продаж, конверсию и другие параметры.
• Закрепление отзывов. Продавец сможет закрепить отзывы покупателей в карточки товаров, ранее эта возможность была только на тарифе «Премиум».
Также «Джем» обеспечит увеличенную частоту опроса по APIWildberries API — это способ получения информации от маркетплейса и выгрузки её в другие сервисы через код. Например, можно автоматически подргужать данные статистики в другой сервис. В нескольких категориях можно будет чаще обращаться к публичному API WB, а выгрузка отчёта «Воронка продаж» будет бесплатной за 365 дней.
