В Минэкономразвития опубликовали коэффициенты-дефляторы, по которым будут рассчитываться отдельные налоги в 2024 году.В связи с новыми показателями лимит для применения УСН индексируют в 1,3 раза, торговый сбор ― в 1,8 раза, а авансовый платёж по НДФЛ увеличится на 5,7%.
Что случилось: каждый год правительство из-за инфляции пересчитывает показатели для расчёта налоговых ставок ― коэффициенты-дефляторы. По ним считают лимит для перехода на УСН, максимальный годовой доход для работы на этой системе и другие сборы в казну.
Какие коэффициенты-дефляторы запланировали на 2024 год:
• 2,4 для НДФЛ;
• 1,284 для налога на дополнительный доход;
• 1,058 для патентной системы;
• 1,329 для УСН;
• 1,828 для торгового сбора.
Какими будут лимиты для УСН:
• 199,35 миллиона рублей для повышения стандартной ставки до 8% на УСН «Доходы» и до 20% УСН «Доходы минус расходы»;
• 265,8 миллиона рублей для потери права на УСН.
Что это значит для бизнеса: увеличатся предельные суммы, с которых предпринимателю нужно будет платить налог по УСН по повышенной ставке или вообще уходить с упрощёнки.
Какие ещё налоги вырастут: торговый сбор увеличится в 1,8 раза, а налог на дополнительный доход и фиксированный авансовый платёж по НДФЛ ― на 5,7%. Подробности о расчётах налогов появятся, когда законопроект будет принят.
